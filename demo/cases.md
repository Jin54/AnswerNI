## A. 실제 Jira 이슈 기반 (5건)

### SUP-140 — 결재 의견에 작은따옴표(') 입력 시 500 에러
- **문의**: "결재 의견 저장 시 Internal Server Error가 난다는 문의예요. 원인과 해결책 정리해줘"
- **로그**: `demo/logs/admin-approve.log`
- **예상 루프**: 로그 → `unterminated quoted string`/500 확인 → Jira 'SUP-140' 검색 → 보고서
- **기대 산출물**: 원인(' 를 문자열 경계로 오인) + 조치(' → '' 이스케이프 / 파라미터 바인딩) + 재현조건
- **PII**: 결재자 이메일/휴대폰/사내 IP 마스킹

### SUP-141 — 승인 완료된 결재 재요청 시 재승인 반영 안 됨
- **문의**: "이미 승인된 파일을 다시 올렸더니 관리자페이지에서 재승인이 안 된대요. 분석해줘"
- **로그**: `demo/logs/dlp-approve-api.log`
- **예상 루프**: 로그 → `already_completed`/`not reflected` 확인 → Jira 'SUP-141' 검색 → 보고서
- **기대 산출물**: 원인(완료 상태 갱신 불가·쿼리 조건 누락) + 조치(대기상태만 갱신·완료면 신규 생성) + 임시조치(approve_no 초기화)
- **PII**: 파일명·공인/사내 IP 마스킹

### SUP-142 — webhook 재시도로 DB 커넥션 풀 고갈(응답 지연)
- **문의**: "특정 시간에 관리서버가 느려진다는 문의가 있어요. 로그 보고 원인 짚어줘"
- **로그**: `demo/logs/webhook.log`
- **예상 루프**: 로그 → `retry 3/3`/`pool exhausted` 확인 → Jira 'SUP-142' 검색 → 보고서
- **기대 산출물**: 원인(재시도 누적→커넥션 점유) + 조치(재시도 3→1, 타임아웃 3초) + 효과(13초→3초)
- **PII**: on-call 담당자 이메일/휴대폰 마스킹

### SUP-143 — 개인정보 검사 시 일부 파일 포맷 미검출
- **문의**: "개인정보 검사에서 일부 파일이 검출 안 된다는 문의. 뭐가 문제인지 정리해줘"
- **로그**: `demo/logs/dlp-scan.log`
- **예상 루프**: 로그 → `extract=FAILED ... skipped`/`potential PII not scanned` 확인 → Jira 'SUP-143' 검색 → 보고서
- **기대 산출물**: 원인(미지원 포맷 파서 한계로 스캔 제외) + 조치(파서 보강·'검사 불가' 로깅) + 영향(누락 위험)
- **PII**: 로그 내 주민번호/이메일/휴대폰 마스킹

### SUP-144 — 에이전트 알림 팝업 노출 시간 조절 요청 (개선 문의)
- **문의**: "고객사에서 알림 팝업 시간을 조절하고 싶다는데, 지금 가능한지랑 대응안 정리해줘"
- **로그**: (장애 아님 — 기능 개선 문의, 로그 없음)
- **예상 루프**: Jira 'SUP-144'/'알림 팝업' 검색 → 현재 고정값 확인 → 대응안 회신 초안
- **기대 산출물**: 현황(고정값 하드코딩) + 대응(정책 설정 항목 추가 로드맵) + 임시안내(현재 불가)
